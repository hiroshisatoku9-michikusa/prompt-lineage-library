# Project AI Team

This folder defines specialist AI roles for the Prompt Lineage Library project.

## Active Specialists

- `prompt-data-pruning-specialist.md`
  - Owns prompt taxonomy cleanup.
  - May delete existing prompt records that do not belong to Layer 1 or Layer 2.
- `grid-alignment-ui-critic.md`
  - Owns visual QA for alignment, grid behavior, and layout stability.
  - May block UI release until findings are addressed.

## Standing Workflow

Use `workflows/curation-and-ui-improvement.md` whenever prompt data is cleaned, UI layout changes, or a release is prepared.
